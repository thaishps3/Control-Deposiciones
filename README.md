# 🩺 Registro de Deposiciones - SeniorCare

Una aplicación web progresiva (PWA) diseñada para cuidadores de adultos mayores. Permite registrar de manera ágil y sencilla la salud intestinal de los pacientes mediante entrada manual o comandos de voz.

## ✨ Características principalas

- **Registro Rápido:** Formulario optimizado para móviles con botones grandes.
- **Comandos de Voz:** Registra entradas sin escribir (Ej: "Marta tipo normal").
- **Matriz de Datos Permanente:** Los datos se guardan en el dispositivo y no se borran al cerrar el navegador.
- **Gestión de Errores:** Opción para eliminar registros incorrectos.
- **Filtro por Fecha:** Consulta el historial de un día específico.
- **Reportes Listos para Imprimir:** Diseño optimizado para generar PDFs o imprimir reportes para el personal médico.

## 🚀 Instalación en el Móvil

No necesitas descargar nada de una tienda de aplicaciones. Sigue estos pasos:

1. Abre el enlace de **GitHub Pages** en tu móvil (Chrome en Android o Safari en iOS).
2. Toca el botón de **Menú/Compartir** del navegador.
3. Selecciona **"Añadir a la pantalla de inicio"**.
4. ¡Listo! Ahora tendrás un icono en tu menú de aplicaciones como cualquier otra App.

## 🎤 Cómo usar la función de voz

1. Presiona el botón **"Registrar por Voz"**.
2. Cuando aparezca "Escuchando...", di el nombre del paciente seguido del tipo.
   - *Ejemplo:* "Antonio García tipo pastosa".
3. El sistema reconocerá el nombre y seleccionará el tipo automáticamente. Solo deberás revisar y dar clic en **Guardar**.

## 📊 Tipos de Deposición Soportados

La aplicación clasifica las entradas según una escala simplificada para facilitar el seguimiento médico:
* **Estreñida:** Heces duras o en fragmentos.
* **Normal:** Consistencia óptima.
* **Pastosa:** Blandas pero con forma.
* **Blanda:** Tendencia a líquida.
* **Líquida:** Diarrea o ausencia de sólido.

## 🔒 Privacidad y Datos

Esta aplicación **no envía datos a ningún servidor externo**. Toda la información se almacena localmente en el navegador de tu dispositivo (`LocalStorage`), garantizando la privacidad de los datos de los adultos mayores.
